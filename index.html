Projects

Book Respiratory System

Developed an advanced C++ book inventory management application, featuring user-friendly functions for adding, editing, and deleting book records. It includes detailed fields like titles, authors, and ISBNs, and utilizes hash tables and trees for efficient data handling and sorting. The system integrates SQLite for data persistence and offers a command-line interface suitable for non-technical users.

Morse Code Converter

Created a Java-based MorseCodeConverter to translate Morse code into English, using a MorseCodeTree for accurate and efficient conversions. It supports direct input and file uploads, processes lines containing Morse code while skipping blanks for batch conversions, and features a visualization of the MorseCodeTree, making it both a practical tool for decoding and an educational resource for Morse code enthusiasts.

GroupMe Bot

The GroupMe Bot leverages the GroupMe API to enrich group chats with interactive features like real-time NFL schedule updates, including team matchups, dates, and times, along with personalized greetings and intelligent date parsing for specific schedule requests. Beyond sharing sports schedules, it enhances user engagement by responding to certain phrases with casual greetings, making it an invaluable and interactive tool for GroupMe users, particularly those keen on NFL events.
